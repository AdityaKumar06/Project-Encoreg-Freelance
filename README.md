
## Project Overview

**EncoreG.com** is the official website for **Encore Global Co., Limited**, a trusted Hong Kong-based provider of premium electronic components and industrial solutions established in 2022.

The site serves global markets in:
- Electronics
- Automotive
- Manufacturing
- Renewable Energy

It is a modern, responsive single-page style website built with clean code and smooth user experience in mind.

**Live Demo:** [https://encoreg.com](https://encoreg.com)

## Technologies Used

- **HTML5** – Semantic structure and modern markup
- **CSS3** – Custom styling with advanced features (flexbox, grid, transitions)
- **Bootstrap 5** – Responsive framework for mobile-first design, grid system, components, and utilities
- **JavaScript** – Interactive features, DOM manipulation, and enhancements
- **EmailJS** – Client-side email sending for the contact form (no backend required)
- **Animations** – Smooth scroll animations, fade-ins, and hover effects using AOS (Animate on Scroll) library or custom CSS/JS

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern hero section with call-to-action
- Service sections highlighting key industries (Electronics, Automotive, Renewable Energy, Manufacturing)
- Contact form powered by EmailJS
- Smooth animations on scroll and interactions
- Fast loading with optimized assets
- SEO-friendly structure
